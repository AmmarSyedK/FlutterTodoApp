# Flutter - Todo App
This is a simple Todo App built using Flutter. With this app, you can add tasks and delete them with a sliding animation.

## Objectives
The objectives of this app are:
-   Provide a simple and intuitive interface for adding and managing tasks.
-   Allow users to quickly and easily delete tasks with a sliding animation.
-   Be lightweight and fast, with minimal dependencies and no unnecessary features.


## Getting Started

To get started with this app, simply download or clone the repository, and open it in your favorite code editor. Then, run the following command to install the required dependencies:
```
flutter pub get
```
After that, you can run the app using the following command:
```
flutter run
```
